# Frontend Pos Management System

A simple ordering system using React, Redux and Tailwinds.

# Required

* Node 16.19.0

## Features

* Basic Stock Management.
* Filter products by categories.
* Creating orders and invoicing created orders.
* Ability to edit and delete products and orders according to authorization status.

## Installation

1. Open terminal, create '.env' using 'env.example'
2. Use npm package manager to install pos management on your computer.
```
3.  npm install
```
```
4.  "npm run start"
```
